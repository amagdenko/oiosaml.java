Building the WS-Federation Extension

This guide explains how to do a checkout from Subversion, build the project and import it into Eclipse. In order to build the WS-Federation extension, it is necessary first to build the OIOSAML.java project itself.

  • Download and install Gant, which is used for building the application
  • Do a svn checkout of oiosaml.java: svn co https://svn.softwareborsen.dk/oiosaml.java/sp/trunk oiosaml
  • Build the OIOSAML.java project: gant publish
  • Do a svn checkout of the WS-Federation extension: svn co https://svn.softwareborsen.dk/oiosaml.java/ws-federation/trunk
  • Build the extension: gant war
  • Import into Eclipse by using File -> Import... -> General -> Existing projects into workspace -> Point to extension checkout dir
